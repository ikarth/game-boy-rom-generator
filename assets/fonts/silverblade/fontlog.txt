FONTLOG for the Silverblade fonts

This file provides detailed information on the Silverblade Font Software.
This information should be distributed along with the Silverblade fonts and any derivative works.

Basic Font Information

Silverblade is a Unicode typeface family that supports most languages that use the Latin script and its variants, and could be expanded to support other scripts.
